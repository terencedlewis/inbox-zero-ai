# Inbox Zero AI

An AI-powered tool that summarizes messages and extracts action items.

## Features

- Summarizes emails into bullet points
- Extracts action items
- Detects deadlines
- Assigns priority

## Future Improvements

- Gmail integration
- Slack bot
- Auto-reply generator
- Task export to Notion

## Coding Guidelines

- Keep functions small and readable
- Use type hints
- Prefer simple UI components (Streamlit)
