# AI Workout Generator

This folder is a standalone Streamlit subproject.

## Run Locally

1. Create and activate a virtual environment in this folder.
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Start the app:

   ```bash
   streamlit run app.py
   ```

## Notes

- Keep dependencies for this app in `ai_workout_generator/requirements.txt`.
- Do not commit virtual environments (`venv/`, `.venv/`).
